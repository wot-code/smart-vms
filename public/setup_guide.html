🛡️ Smart VMS: Professional Setup Guide

This document provides the necessary steps for developers and system administrators to deploy the Smart Visitor Management System (Smart VMS) in a local environment using XAMPP.

1. 🚀 Technical Requirements

Ensure the following software is installed on the host machine:

  • XAMPP Control Panel (equipped with PHP 8.2 or higher).
  • Composer (The dependency manager for PHP).
  • Modern Web Browser (Google Chrome or Safari recommended for PWA features).

2. 🛠️ Installation Steps

Step 1: Project Extraction

Extract the project files or clone the repository into your C:\xampp\htdocs\ directory:

cd C:\xampp\htdocs\smart-vms

Step 2: Dependency Installation

Open your terminal/command prompt inside the smart-vms folder and run:

composer install

Step 3: Environment Configuration

Locate the .env.example file in the root folder. Rename it to .env and update the database credentials:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=vms_db
DB_USERNAME=root
DB_PASSWORD=

Step 4: Database Setup

  1. Start Apache and MySQL in XAMPP.
  2. Open http://localhost/phpmyadmin.
  3. Create a new database named vms_db.
  4. Select the database and click "Import".
  5. Upload the file: vms_db_dump.sql found in the project root.

Step 5: Initialization

Generate the unique application security key:

php artisan key:generate

3. 🏃 Running the Application

To launch the system, execute the following command:

php artisan serve

The application will be accessible at: http://127.0.0.1:8000

4. 🔑 Default Access

Once the database is imported, you can log in using the administrator accounts provided. All visitors can access the public registration portal via the /checkin route.
